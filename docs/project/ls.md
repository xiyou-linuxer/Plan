# 实现自己的 ls 命令

## 知识要点

1. Linux 系统编程目录和文件部分
2. C 语言编程能力

## 任务说明

`ls` 是 GNU/Linux 操作系统中常见的命令。
请使用 C 语言，基于 GNU/Linux 操作系统，编程实现 `ls` 命令的**部分**功能。

## 验收要求

- 实现 ls 的 -a、-l、-R、-t、-r、-i、-s 参数，并允许这些参数任意组合。
- `-R` 需要对 `/` 的遍历测试。
- 界面美观（输出对齐、带颜色显示等）。
- 无资源与内存泄漏。
- 开发过程记录、总结并发布在个人博客中。

## 提交方式

- 在任务截止时间之前将代码上传到个人仓库。
- 截止时间：2024.02.17 24:00 (UTC+8)

## 命令输入示例

```bash
ls -a -l
ls -a /home
ls -alR /
ls -t
ls -ai -t /home
ls -i /home /etc
```

注：
命令输出示例请参考 `ls` 命令。
