### Linux系统编程

学习 Linux 系统编程，就是学习使用 Linux 内核和 C 库（通常是 glibc，它封装了 Linux 内核提供给我们的系统调用及其他有用的工具函数）为我们的编写应用程序提供的接口。
在 Linux 运行的各种程序都会直接或间接的调用 Linux 系统调用（或者 C 库）。
学会 Linux 系统编程，我们才可以实现能在 Linux 上运行的特定功能的程序。
